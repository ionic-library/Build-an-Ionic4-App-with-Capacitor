# Build a mobile app  with Ionic Capacitor

Read the post in my <a href="http://blog.via-internet.de/en/blog/2018/03/16/ionic-3-starting-with-capacitor/">blog</a>


## How to setup the environment and code

    npm -g install ionic

    ionic start SampleApp super  --no-link

    cd SampleApp

    npm install --save @capacitor/core @capacitor/cli

    npx cap init sample-app com.example.sampleapp
    npm install -save @capacitor/core
    npx cap add ios

    npm run build

    npx cap copy
    npx cap copy ios
    npx cap open ios
